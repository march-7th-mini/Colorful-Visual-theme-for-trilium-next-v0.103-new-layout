### ✨ Quick Overview

| Feature | Description |
|---|---|
| 🎨 **VOCALOID Heading Styles** | Colorful headings (h2-h6) with 3 style variants |

| 📝 **Blockquote Colors** | VOCALOID-themed blockquote backgrounds |
| 🔢 **List Marker Colors** | Multi-level colored list markers |

| ☀️🌙 **Theme Toggle** | One-click light/dark switch |
